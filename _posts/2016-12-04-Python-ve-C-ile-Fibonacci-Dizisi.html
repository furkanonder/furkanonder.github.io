Kısaca fibonacci dizisini tanıyalım:
Fibonacci dizisi her sayının kendinden önceki sayı ile toplanması sonucu oluşan bir sayı dizisidir.


n yerine sırasıyla 0,1,2,3,4,5,6,7... şeklinde değerler verdiğimizde 0,1,1,2,3,5,8,13…şeklinde bir fibonacci dizisi elde ederiz.Fibonacci dizisi ile ilgili daha ayrıntılı bilgi almak için burayı ziyaret edebilirsiniz.

Şimdi fibonacci dizisi için python ve c ile yazmış olduğum kodları inceleyelim;
C: 
//Created by Furkan Taha ÖNDER on 26.11.2016.
#include <stdio.h> 
int main(){
   int n=10, a = 0, b = 1, deger;
     
   //Bir döngü oluşturuyoruz.Bu döngü b değeri n den büyük olana kadar tekrar edecek.
   for( ; b < n; ){
     deger = a+b;
     a = b;
     b = deger;
     printf("%d\n",b);
    }
    
  getchar();
  return 0;  
}
